Build a modern, mobile-first workout and nutrition web app in Bubble. The app should help users track daily workouts, record exercises, and monitor fitness progress in a simple but premium-looking interface. Use a dark theme with creative accent colors, smooth gradients, rounded cards, clean typography, and an overall sleek gym/fitness style. The UI should feel fast, modern, and highly interactive with smooth page transitions, animated buttons, subtle hover effects, and a small click sound when users tap important actions.

The app should include secure sign up/login and a user profile setup with name, age, gender, height, weight, fitness goal, and experience level. The main workout section should let users choose a muscle group, select an exercise, and log sets, reps, and weight used for each session. Save workout history so users can view past workouts and compare performance over time. Add a smart progression system where, after the user logs an exercise for the first time, the app stores that weight and later suggests the ideal next weight for the same exercise and muscle group based on previous performance. If the user completes all reps successfully, suggest a small weight increase; if not, suggest keeping the same weight or lowering slightly. Show this suggestion clearly before the next workout.

Create a dashboard that shows total workouts completed, weekly streak, calories burned, recent exercises, personal records, and progress charts for strength growth. Add a progress section with body weight tracking, measurement logging, and before/after photo uploads. Include an exercise library with common exercises, muscle group tags, and short instructions so users can learn proper form.

Also create a diet section where users can get a customized meal plan based on weight, height, gender, activity level, and goals like weight loss, weight gain, or maintenance. The app should calculate daily calories, protein, carbs, and fats, then generate a simple meal plan for breakfast, lunch, snacks, and dinner. Let users edit meals and track daily food intake.

Add extra useful features like workout reminders, water reminders, achievement badges, streak rewards, favorites, search/filter by muscle group, and a clean profile/settings page. The final product should feel like a complete fitness companion app that is easy to use, visually premium, and motivating for daily workouts and diet tracking.
