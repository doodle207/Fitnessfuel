-----------------------------------

- India → Razorpay
- Outside India → Stripe

2) Razorpay

- POST /api/payments/razorpay/verify

- Verify signature before unlocking premium

3) Stripe
- POST /api/payments/stripe/create-session
- POST /api/webhooks/stripe
- Handle:
  - checkout.session.completed
  - invoice.paid
- Verify webhook signature

5) Database

- userId

- country
- isPremium
- paymentProvider
- planName
- subscriptionId
- paymentStatus
- expiryDate

6) Premium unlock
- Only unlock via backend verification/webhooks
- Never rely on frontend only
